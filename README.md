# Classting 과제

> 클래스팅의 채용 과제를 위한 프로젝트 입니다.

## start

```
1. yarn
2. yarn start
```

## 사용 라이브러리

- styled-components
- eslint
- prettier
- react-vis
- react-hook-form
- react-timer-hook
